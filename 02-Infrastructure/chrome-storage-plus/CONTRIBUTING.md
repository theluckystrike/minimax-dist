# Contributing to chrome-extension-starter-mv3

We welcome contributions! Please follow these guidelines.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/chrome-extension-starter-mv3.git`
3. Install dependencies: `npm install`

## Making Changes

1. Create a branch: `git checkout -b feature/your-feature`
2. Make your changes
3. Test locally with `npm run dev`
4. Commit your changes: `git commit -m "Add feature: your feature"`
5. Push to GitHub: `git push origin feature/your-feature`
6. Open a Pull Request

## Code Style

- Use TypeScript
- Follow existing code patterns
- Add comments for complex logic

## Documentation

- Update README.md for user-facing changes
- Add JSDoc comments for new APIs

## Questions?

- Open an issue for bugs or feature requests
- For general questions, reach out on Twitter @zovotools
