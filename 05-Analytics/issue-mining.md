# Issue Mining - Feedback Collection

## Categories

### Feature Requests
- Look for: "would be nice", "feature", "add support"
- Action: Add to roadmap

### Bug Reports
- Look for: "bug", "error", "doesn't work", "broken"
- Action: Priority fix

### Questions
- Look for: "how to", "can i", "is it possible"
- Action: Improve docs

### Pain Points
- Look for: "hard", "difficult", "frustrating", "annoying"
- Action: Address in content
