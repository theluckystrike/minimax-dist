# UTM Configuration

## Tracking Structure

```
utm_source    → Where the click came from
utm_medium    → Type of content
utm_campaign  → Specific repo/project
utm_content   → Specific placement
```

## UTM Parameters by Channel

### GitHub
```
?utm_source=github
&utm_medium=readme
&utm_campaign=chrome-extension-starter-mv3
&utm_content=top-navigation
```

### npm
```
?utm_source=npm
&utm_medium=package-page
&utm_campaign=chrome-extension-starter-mv3
&utm_content=package-description
```

### Dev.to

&utm_medium=article
