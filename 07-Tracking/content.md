## Platform Guidelines

### Dev.to
- **Max tags:** 4
- **Recommended tags:** javascript, webdev, opensource, chrome
- **Series:** Group related posts
- **Canonical URL:** Point to your blog
- **Cover image:** 1280x640, branded

### Hashnode
- **Blog domain:** Use personal for SEO
- **GitHub backup:** Enable
- **Newsletter:** Enable
- **Tags:** Different from Dev.to

### Medium
- **Publications:** JavaScript in Plain English, Better Programming
- **Friend links:** Include for paywall bypass
- **Tone:** Less technical, more story-driven

### Twitter/X
- **Thread length:** 8-12 tweets
- **Hook:** First tweet must grab attention
- **CTA:** Always include link
- **Hashtags:** 1-2 max, relevant

### Hacker News
- **Title format:** "Show HN: [Tool] – [value prop]"
- **Timing:** Tuesday-Thursday, 9-11 AM EST
- **Comments:** Respond within 2 hours
- **Frequency:** Max 1 per 2 weeks

### Reddit
- **Subreddits:** r/webdev, r/javascript, r/chrome_extensions
- **Rules:** Read sidebar before posting
- **Karma:** Build up before self-promoting
- **Ratio:** 10 helpful comments per self-promotion

## UTM Link Template

For each article, create these UTM links:

```
Article: [Title]
URL: https://example.com/article

Extension: [Extension Name]
UTM Base: ?utm_source=[platform]&utm_medium=article&utm_campaign=[repo-name]

Full Link: https://chromewebstore.google.com/detail/[ext-id]?utm_source=[platform]&utm_medium=article&utm_campaign=[repo-name]
```
